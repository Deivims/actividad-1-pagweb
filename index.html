Actividad de Investigación: Elementos de Páginas Web

Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

  1. Web 2.0

    La Web 2.0 se refiere a la segunda generación de la web, caracterizada por un enfoque en la colaboración y el intercambio de información. Se centra en aplicaciones web que permiten la interacción de los usuarios, como redes sociales, wikis y blogs.

  2. Web 3.0

    La Web 3.0, también conocida como la web semántica, busca proporcionar un significado a los datos en la web para mejorar la interacción entre los usuarios y las máquinas. Utiliza tecnologías como el RDF y ontologías para hacer que la web sea más inteligente y conectada.

  3. La W3C y los estándares

    La World Wide Web Consortium (W3C) es una organización que desarrolla estándares y directrices para mejorar la calidad de la web. Su objetivo es asegurar que la web sea accesible, interoperable y útil para todos los usuarios.

II. Lenguajes de etiquetas

  1. SGML

    Standard Generalized Markup Language (SGML) es un estándar para definir lenguajes de marcado que describe la estructura de documentos. HTML y XML son lenguajes derivados de SGML.

  2. XML

    Extensible Markup Language (XML) es un lenguaje de marcado que define un conjunto de reglas para codificar documentos en un formato legible por humanos y máquinas. Se usa para transportar y almacenar datos.

  3. HTML

    Hypertext Markup Language (HTML) es el lenguaje estándar para crear y diseñar páginas web. Utiliza una serie de etiquetas para estructurar el contenido en la web.

  4. XHTML

    Extensible Hypertext Markup Language (XHTML) es una versión más estricta y reformulada de HTML basada en XML. Introduce reglas más rigurosas para asegurar la correcta estructura de los documentos.

  5. HTML5

    HTML5 es la última versión del HTML y ofrece nuevas características y elementos que permiten el desarrollo de aplicaciones web más avanzadas, como video, audio, y gráficos 2D y 3D.

III. Otros lenguajes relacionados al desarrollo web

  1. CSS/CSS3

    Cascading Style Sheets (CSS) es un lenguaje de diseño para la presentación de documentos HTML. CSS3 es la última versión, que incluye nuevas características como transiciones, transformaciones y animaciones.

  2. JavaScript

    JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Se utiliza para crear efectos, manejar eventos y modificar el contenido en tiempo real.

  3. Ajax

    Asynchronous JavaScript and XML (Ajax) es una técnica que permite actualizar partes de una página web sin tener que recargar toda la página. Facilita la creación de aplicaciones web interactivas.

  4. Tecnologías y lenguajes del lado del servidor

    Estas tecnologías, como PHP, Python y Ruby, se ejecutan en el servidor para procesar datos y generar contenido dinámico antes de enviarlo al navegador del cliente.

  5. Navegadores

    Los navegadores web, como Chrome, Firefox y Safari, son aplicaciones que permiten a los usuarios acceder y visualizar páginas web. Cada navegador puede interpretar y mostrar el contenido web de manera ligeramente diferente.

  6. Motores renderizados

    Los motores de renderizado son componentes de los navegadores que interpretan el código HTML, CSS y JavaScript para renderizar y mostrar las páginas web en la pantalla.

IV. Herramientas de desarrollo

  1. Software WYSIWYG

    What You See Is What You Get (WYSIWYG) es un tipo de software que permite a los diseñadores crear páginas web visualmente, sin necesidad de escribir código manualmente. Ejemplos incluyen Adobe Dreamweaver y Microsoft Expression Web.

  2. Otras alternativas

    • Figma: Una herramienta de diseño colaborativo para la creación de interfaces y prototipos.
    • InVision: Plataforma para el diseño y prototipado interactivo, así como para la colaboración en equipo.
    • Zeplin: Herramienta que facilita la transferencia de diseño entre los diseñadores y desarrolladores, proporcionando especificaciones detalladas.
    • GitHub: Plataforma de desarrollo colaborativo que utiliza Git para el control de versiones y la gestión de código fuente.

  3. Inspección de código desde el navegador

    Los navegadores modernos permiten inspeccionar y depurar el código HTML, CSS y JavaScript de una página web a través de las herramientas de desarrollo integradas.

Tema 2: El Diseño De Una Página Web

I. Tipos de páginas Web

Existen varios tipos de páginas web, entre ellas:

  • Páginas estáticas: Contenidos fijos que no cambian en respuesta a las acciones del usuario.
  • Páginas dinámicas: Contenidos generados en tiempo real, generalmente mediante interacciones con bases de datos.
  • Páginas interactivas: Ofrecen una experiencia de usuario enriquecida mediante la incorporación de elementos interactivos y multimedia.
  • Páginas de e-commerce: Diseñadas para el comercio electrónico, permitiendo la compra y venta de productos en línea.

II. Puntos básicos en la elaboración de una página Web

Para elaborar una página web, es fundamental considerar los siguientes puntos:

  • Definición del propósito y objetivo: Conocer el objetivo principal de la página web y el público objetivo.
  • Estructura y diseño: Crear una estructura clara y un diseño atractivo y funcional.
  • Contenido: Generar contenido relevante y de calidad que sea útil para los usuarios.
  • Optimización para motores de búsqueda (SEO): Implementar prácticas de SEO para mejorar la visibilidad en los motores de búsqueda.
  • Pruebas y revisión: Realizar pruebas para asegurar que la página web funciona correctamente en diferentes dispositivos y navegadores.

Tema 3: Estructura De Una Página Web

I. Jerarquía de directorio de un proyecto web

La jerarquía de directorios de un proyecto web organiza los archivos de manera estructurada. Un esquema típico incluye:

  • /index.html: Página principal.
  • /css/: Carpeta que contiene archivos CSS.
